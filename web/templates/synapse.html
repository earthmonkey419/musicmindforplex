What this does: Synapse analyzes the actual audio in your library — not metadata, not AI guesses — using Essentia to measure tempo (BPM), musical key, and danceability directly from the waveform, and (if the voice/instrumental model file has been downloaded — see the repo README) also verifies each track's real instrumental status the same way, in the same pass. It's completely free (no API calls). The first full-library run is the only slow one — it can take a couple of days depending on your library size and hardware, and is safe to run in the background; just start it and check back later. After that initial backlog is cleared, Synapse runs automatically as the final step of every Full Sync, analyzing only genuinely new tracks — typically seconds to minutes, not days. The manual "Run Now" button below is for that one-time backlog clear (or to re-analyze on demand); day-to-day, you shouldn't need to touch it.

📊 Count Audio File Types

Walks your full library and shows a breakdown of file formats (MP3, FLAC, etc). Read-only — no analysis, no writes. Fast.

⏱️ Estimate Time

Analyzes a few real sample tracks live, on this hardware, to project how long a full run would actually take for your library. No permanent writes — takes about a minute.

🔬 Limited Run

Runs the real analysis, but capped to a small number of tracks — a safe way to confirm everything works before committing to the full multi-day job.

🚀 Full Run

Analyzes every track in your library and stores the results. Resumable — safe to stop and restart anytime, already-analyzed tracks are skipped. Recommend running Estimate Time first so you know what to expect.

This job runs fully detached from the web app — it survives app restarts and doesn't need this page to stay open. Check progress anytime via the status banner above, or click "Refresh Log" below for recent detail.

⚠️ Errors

Tracks Synapse and/or VI couldn't analyze — corrupted files, unreadable formats, or files that timed out. Safe to ignore; these are skipped automatically on future runs.
